# Stocks-Live-Scraper
Overview: a.This Python project fetches live stock prices for selected companies using yfinance and stores them in CSV. | b.Tracked stocks: AAPL, MSFT, GOOGL, AMZN, TSLA, NVDA, META, BRK-B, JPM, V

Features: a.Live data scraping from yfinance | b.Timestamped output for tracking | c.Historical CSV storage for trend visualization | d.Automated fetching at configurable intervals (default: 1 hour)

Requirements: a.Python 3.8+ | b.yfinance library

Install dependencies: pip install yfinance

Notes: a.1-hour interval captures meaningful price changes for visualization. | b.You can adjust the interval for faster or slower updates.
